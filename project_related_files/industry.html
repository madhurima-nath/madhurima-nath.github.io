GenAI & LLM Applications

A retrieval-augmented generation (RAG) architecture was used across three client engagements to build question-answering systems over domain-specific document collections. Each deployment adapted the same core pipeline — document ingestion, embedding, retrieval, and LLM generation — to the requirements and compliance constraints of the sector.

  • Financial services. Built a product discovery system for a major financial institution on Databricks, integrating Llama with a LangChain agent framework and MLflow pipelines. Coordinated with compliance teams throughout development. Delivered measurable improvements in information retrieval across the institution's product catalogue.
  • Equipment rental. Deployed a knowledge support tool for a leading equipment rental firm to assist sales representatives with customer inquiries, reduce manual document searches, and support internal training. Achieved a 22% reduction in customer inquiry response time.
  • Healthcare. Built a Q&A system for a biopharma and medical technology client using Claude on AWS Bedrock to extract insights from clinical and technical datasets.

NLP Pipelines

Three NLP engagements applying text classification, sentiment analysis, and topic modelling to domain-specific document collections. Each addressed a different operational problem — product taxonomy harmonisation, safety insight extraction, and legal document review — using models suited to the data volume and task.

  • Global procurement — CPG. Designed and built an end-to-end Azure Databricks solution for a global consumer goods company managing a $34B spend portfolio across multiple ERP systems with no common taxonomy. NLP text classification was used to harmonise product categories across systems. Delivered with MLflow-integrated pipelines and a CI/CD framework; accelerated the client's data foundation roadmap by approximately one year.
  • Safety analysis — energy utility. Built NLP pipelines on Azure Databricks to extract safety-related themes from incident reports using sentiment analysis and topic modelling, with automated weekly reports and dashboards for the executive safety committee.
  • Document review — legal services. Developed a document classification system for e-discovery using Word2Vec embeddings and SVM classifiers on AWS SageMaker, with an automated retraining pipeline for incremental model updates. Reduced manual review hours and associated costs.

Analytics & Data Platforms

Network predictive maintenance — telecommunications. Developed a statistical device health scoring system for a telecom client to proactively identify at-risk network infrastructure before service disruption. Hourly radio frequency metrics (signal-to-noise ratio, downstream receiving power, upstream transmitting power) were combined into a weighted score; DBSCAN spatial clustering then identified geographic concentrations of high-scoring devices pointing to faulty infrastructure nodes rather than individual device issues. Automated SQL procedures ran daily across a 90-day rolling window. The approach successfully flagged known problematic nodes with high precision in the validation region, and was subsequently extrapolated nationwide. Projected savings of $8M+ annually through reductions of approximately 30,000 customer support calls and 6,000 service visits. Full detail →

Validated large-scale data quality checks and transformation rules across all data resources contributing to patient health information sharing.
